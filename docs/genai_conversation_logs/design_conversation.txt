Hotel Risk & Operations Intelligence API
Final Project Specification

1. Final Project Summary
The Hotel Risk & Operations Intelligence API is a RESTful backend system that combines historical booking data from the Hotel Booking Demand dataset with user-generated operational data to support hotel risk monitoring and operational analysis. The historical dataset provides information about booking behaviour, cancellations, demand patterns, and customer segments, while the API allows staff to record operational incidents and internal manager notes linked to specific bookings. By combining these two layers of data, the system can expose analytical endpoints that reveal operational patterns such as cancellation trends, incident frequency, and high-risk bookings. This design transforms a static dataset into an operational intelligence system rather than a simple data viewer or review platform.
